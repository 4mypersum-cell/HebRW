# מערכת לימוד עברית - Hebrew Learning App
## ללמוד עברית! Learn Hebrew Reading & Writing

---

## 📚 תיאור הפרויקט | Project Description

מערכת אינטראקטיבית ללימוד קריאה וכתיבה בעברית למתחילים, עם גיימיפיקציה מלאה, תרגילים אינטראקטיביים, ומעקב אחר התקדמות.

An interactive system for learning Hebrew reading and writing for beginners, with full gamification, interactive exercises, and progress tracking.

---

## ✨ תכונות עיקריות | Main Features

### 🎮 גיימיפיקציה | Gamification
- מערכת ניקוד (Points System)
- רמות (Levels)
- Streak של ימים רצופים (Daily Streak)
- מעקב זמן למידה יומי (Daily Study Time Tracking)
- הישגים מיוחדים (Special Achievements)

### 📖 תכנים לימודיים | Learning Content
- **22 אותיות האלף-בית העברי** - 22 Hebrew Alphabet Letters
- **14 מילים פשוטות** - 14 Simple Words
- **צלילים ושמות של אותיות** - Letter Sounds and Names
- **תרגום אנגלי** - English Translation

### 🎯 תרגילים אינטראקטיביים | Interactive Exercises
- **תרגיל "מצא את האות"** - "Find the Letter" Exercise
  - זיהוי אותיות בקבוצה מעורבת
  - משוב מיידי עם צבעים
  - ניקוד על תשובות נכונות

- **תרגול קריאת מילים** - Word Reading Practice
  - הצגת מילה בעברית
  - אפשרות לראות תרגום
  - לימוד קריאה מימין לשמאל

- **בדיקה יומית** - Daily Quiz
  - 5 שאלות רנדומליות
  - בחירה מרובה
  - ניקוד וסיכום

### 🔊 תכונות נוספות | Additional Features
- **הקראת טקסט** - Text-to-Speech
  - כפתור רמקול בכל מקום
  - הגייה בעברית
  - קצב למידה מותאם

- **מעקב התקדמות** - Progress Tracking
  - אותיות שנלמדו
  - נקודות שנצברו
  - זמן למידה יומי
  - מטרה של 30 דקות ביום

## 🚀 איך להתחיל | How to Start

### דרך 1: פתיחה מהירה (קל ביותר!)
1. פתח את הקובץ `hebrew-app-simple.html`
2. גרור אותו לדפדפן (Chrome, Firefox, Safari, Edge)
3. התחל ללמוד! 🎉

### דרך 2: הרצה מקומית
1. הורד את כל הקבצים
2. פתח את `hebrew-learning-app.html` בדפדפן
3. (דורש חיבור אינטרנט לספריות חיצוניות)

### דרך 3: שרת מקומי (למפתחים)
```bash

python -m http.server 8000

npx http-server

# לאחר מכן פתח בדפדפן:
# http://localhost:8000/hebrew-app-simple.html
```

## 🎯 מדריך שימוש | User Guide

### התחלה
1. הכנס את שמך במסך הבית
2. בחר באחד מהמודולים:
   - **לימוד אותיות** - למד כל אות וצליל
   - **תרגול מילים** - תרגל קריאת מילים
   - **בדיקה יומית** - בדוק את הידע שלך
   - **התקדמות שלי** - ראה סטטיסטיקות

### מסך לימוד אותיות
1. צפה באות הגדולה
2. לחץ על 🔊 לשמיעת ההגייה
3. למד את שם האות והצליל
4. לחץ "התחל תרגיל" לתרגול
5. מצא את כל האותיות הזהות
6. לחץ "בדוק" לקבלת משוב
7. עבור לאות הבאה

### תרגיל "מצא את האות"
- **לחץ** על האותיות שאתה חושב שנכונות
- **ירוק** = בחרת נכון ✅
- **כתום** = פספסת אות נכונה ⚠️
- **אדום** = בחרת אות לא נכונה ❌
- **נסה שוב** = תרגיל חדש עם אותיות מעורבבות

### מסך תרגול מילים
1. צפה במילה העברית
2. לחץ 🔊 לשמיעת המילה
3. נסה לקרוא בעצמך
4. לחץ "הצג תרגום" לבדיקה
5. עבור למילה הבאה

### בדיקה יומית
1. ענה על 5 שאלות
2. בחר את הצליל הנכון לכל אות
3. קבל ניקוד בסוף
4. נסה להשיג 5/5! 🏆

## 💡 טיפים ללמידה | Learning Tips

### עברית 101
- עברית נקראת **מימין לשמאל** ←
- רוב האותיות זהות בתחילת ובסוף מילה
- יש אותיות סופיות מיוחדות (ך, ם, ן, ף, ץ)
- ניקוד עוזר לדעת את התנועות

### איך ללמוד יעיל
1. **תרגל כל יום** - אפילו 10-15 דקות
2. **השתמש ב-🔊** - שמע את ההגייה הנכונה
3. **חזור על אותיות קודמות** - חזרה היא אמא הלמידה
4. **תרגל עם תרגיל "מצא את האות"** - זה משפר זיהוי חזותי
5. **אל תמהר** - קח את הזמן ללמוד כל אות היטב
